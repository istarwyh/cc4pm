# lawyer-case-intake

读取案件材料，生成案件摘要、事实时间线、材料缺口和客户追问清单。

## 输出要求

- 案件摘要
- 当事人信息
- 时间线
- 涉及金额
- 已有材料
- 缺失材料
- 需要追问客户的问题
- 不确定事项

## 工作规则

关键事实必须标注来源文件。没有来源的事实标注【材料未见】。不要补充材料中不存在的事实、金额、日期或主体信息。

## 免责声明

本 Skill 仅用于教学和律师工作辅助，不构成法律意见。输出为律师审阅初稿，最终由律师复核。
