# SaaS 软件服务合同纠纷模拟案例

本目录为 cc4Lawyer 课程使用的虚构材料，用来串联 Claude Code 项目目录、CLAUDE.md、上下文管理、Skills、Agents、Supervisor、Reviewer/Evaluator 和交付包。所有主体、人员、金额和事实均为教学模拟，不对应真实公司或个人。

## 案件背景

甲方 A 公司与乙方 B 公司签署《软件服务合同》，约定 A 公司为 B 公司提供企业内部 AI 知识库系统，合同金额 30 万元。B 公司已支付首付款 10 万元，剩余 20 万元未支付。B 公司主张系统未达到验收标准，拒绝支付尾款。A 公司认为已完成交付，且 B 公司实际使用系统超过 3 个月，应支付尾款并承担违约责任。

## 材料清单

- 软件服务合同
- 补充协议
- 交付确认邮件
- 微信聊天记录
- 发票与付款记录
- 系统登录记录
- 客户投诉邮件
- 催款函
- 对方律师函

## 示例输出

`expected-outputs/` 是课程讲解用的课堂样稿，用来展示一次垂直行业课程如何形成摘要、时间线、材料对齐表、复核清单、客户汇报草稿和可复用交付包。

这些文件不是法律行业标准答案，也不构成法律意见。真实项目中必须由律师团队按自己的 SOP、专业判断和复核标准重写后才能对外使用。
