# 可复用客户交付包清单（教学示例）

> 本清单用于课程讲解“Package”阶段。实际交付前需移除内部过程材料，并由律师复核。

## 一、客户可见交付件

- 案件摘要
- 事实时间线
- 证据目录
- 合同关键条款与风险审查表
- 对方抗辩及我方回应思路
- 诉讼准备报告
- 材料补充清单
- 客户汇报邮件

## 二、内部工作底稿

- 材料索引
- Agent Team 分工记录
- Risk Reviewer 复核报告
- 提示词和工作流记录
- 复盘沉淀笔记

## 三、不得进入客户版本的内容

- Claude、AI 提示词、内部 Agent 名称。
- 未经律师确认的法律结论。
- 未脱敏个人信息、商业秘密或未公开交易资料。
- “必胜”“稳赢”“法院一定支持”等绝对化表达。

## 四、交付前检查

1. 主体名称、金额、日期是否一致。
2. 每个关键事实是否有材料来源。
3. 不利事实是否被如实提示。
4. 诉讼请求和法律依据是否由律师确认。
5. 客户版本是否专业、克制、清晰。
