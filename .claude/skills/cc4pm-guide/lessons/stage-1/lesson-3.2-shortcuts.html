命令 / 快捷键 作用
tmux new -s <name>新建命名会话
tmux ls列出所有会话
tmux attach -t <name>重新连接到会话
Ctrl+B → D断开当前会话（后台保留）
Ctrl+B → %左右分屏
Ctrl+B → "上下分屏
Ctrl+B → 方向键切换面板
Ctrl+B → C新建窗口
Ctrl+B → N / P下一个 / 上一个窗口
exit关闭当前面板或窗口
