当世界名著遇见大模型

五部 AI 时代的文学经典，献给每一个和 Token 搏斗的人

如果有一天，AI 读完了人类所有的文学经典，它会怎样理解自己的处境？

或者换个问法——如果那些伟大的作家活在今天，面对 GPT、Claude、Gemini 轮番登场的世界，他们会写下什么样的开场白？

我们替他们写了。

第 一 部

《 Token 苍穹 》

「三个月河东，三个月河西，莫欺 Token 少。」

——《Token 苍穹》

每个月总有那么几天，Token 余额见底，Claude 跟你说「请明天再来」，ChatGPT 把你踢回 3.5。你蹲在免费层里刷新页面，眼睁睁看着付费用户在朋友圈晒 Opus 的输出，心想：凭什么。

更扎心的是鄙视链。Max 订阅用户看不起 Pro，Pro 看不起免费，免费的看不起还在用国产套壳的。有人一天烧掉你一个月的额度，发条推说「也就那样吧」。你攥着最后 5 条消息，字斟句酌，生怕浪费一个 Token——这种感觉，像极了月底看银行余额。

但别急。三个月河东，三个月河西。上个季度你还在为 GPT-4 的天价 Token 肉疼，这个季度价格已经砍了 90%。今天你没有的 Token，过几天就会有。莫欺 Token 少——这不是安慰，这是这个行业唯一靠谱的规律。

第 二 部

《了不起的 OpenAI》

「每逢你想要批评大模型的时候，你就记住，这个世界上所有的大模型，并不是个个都有过 OpenAI 拥有的那些训练集。」

——《了不起的 OpenAI》

菲茨杰拉德笔下的盖茨比凝望着对岸的绿灯，而 Sam Altman 凝望的是下一轮融资。

不是每个模型都含着金钥匙出生。有的模型从小喝的是整个互联网的语料，有的只能啃几本教科书长大。你说开源模型不如闭源？那是因为人家从小上的就是贵族学校，吃的就是 RLHF 加量套餐。

所以，下次骂一个开源模型笨的时候，想想它经历了什么。

第 三 部

《Vibe Coding》

「幸福的 Prompt 总是相似的，不幸的 Prompt 各有各的不幸。」

——《Vibe Coding》

托尔斯泰要是活到今天，大概会成为 Prompt Engineering 领域的顶级博主。

好用的 Prompt 都长一个样：角色设定清晰、任务描述具体、输出格式明确、还附带几个 few-shot 示例。而那些让 AI 胡说八道的 Prompt，各有各的离谱——有人忘了给上下文，有人自相矛盾，有人一句话要 AI 同时写代码、画图、唱歌、做饭。

Vibe Coding 的精髓就是：别跟 AI 较劲，跟着感觉走。写不出来？换个说法。还写不出来？再换个模型。人生苦短，何必跟一个概率模型死磕。

第 四 部

《Token 神话》

「真正严肃的 Coding 问题只有一个：没有 Token。」

——《Token 神话》

加缪说真正严肃的哲学问题只有一个，就是自杀。放到 AI 编程领域，真正严肃的问题也只有一个——Token 用完了。

你可以忍受 AI 偶尔犯蠢，可以忍受它幻觉，可以忍受它把 Python 写成 Java。但当屏幕弹出「Rate limit exceeded」的那一刻，一切存在主义的荒谬感扑面而来。

西西弗斯推着石头上山，我们推着 Prompt 撞 Token 墙。唯一的区别是，西西弗斯不用付费。

第 五 部

《百年 AI》

「多年以后，面对 AI 生成的一百万字小说，你将会回想起 Google 带你去见识 Word2Vec 的那个遥远下午。那时，上下文窗口只有 512 个 Token，人类还相信故事是属于自己的。」

——《百年 AI》

马尔克斯式的开头，配上 AI 编年史，竟然毫无违和感。

2013 年，Mikolov 发表 Word2Vec 论文的时候，没人想到这个把词变成向量的小把戏，会在十年后催生出能写小说、能编代码、能跟人吵架的通用智能。那时候 512 个 Token 的上下文窗口已经算奢侈品，而今天 Claude 的窗口够装下一整本《百年孤独》还有余。

人类曾以为创作是最后的护城河。现在这条河上，AI 已经架了桥、修了路、还开了收费站。

尾 声

写在最后

这五句话好笑，是因为它们说的全是真的。

大模型的迭代速度确实三个月一个世代，训练数据确实决定了模型的天花板，好的 Prompt 确实都长一个样，Token 确实是最稀缺的资源，而 AI 确实已经在接管我们以为只属于人类的东西。

文学的力量在于，它能把复杂的现实压缩成一句话。这五句话，就是 AI 时代的五个切片。
