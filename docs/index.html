FAQ

常见问题

用之前你可能想知道的事

不需要。cc4pm 面向三类人：独立创始人、产品经理和开发者。BMM（业务建模）和 CIS（创意智能）模块完全不需要编程基础，只要你会用 Claude Code 发消息就能上手。如果你是开发者，工程工具链（TDD、代码审查、安全扫描）会额外提升你的效率。

直接问 AI 就像找一个聪明但什么都不了解你项目的人临时帮忙——你每次都要从头解释背景，输出质量取决于你的 Prompt 水平。

cc4pm 相当于给 Claude Code 装上了一套专为产品场景调优的大脑：200+ 个结构化 Skill 定义了每个任务的最佳执行方式，BMAD 方法论确保每个步骤的输出都能无缝衔接下一步，持续学习机制让它越来越了解你的项目。

安装通常在 5 分钟内完成。克隆仓库后打开 Claude Code，输入 /cc4pm-guide，AI 私教会引导你完成整个环境配置，并提供 26 节交互式课程（总计约 2 小时）。

你的第一步可以是：告诉 cc4pm 你的产品想法，它会帮你做市场调研并生成第一份产品简报——这通常在 30 分钟内完成。

cc4pm 原生支持 Claude Code（最完整功能），同时适配 Cursor、OpenAI Codex 和 OpenCode。安装脚本会自动检测你使用的工具并选择对应的安装目标，无需手动配置。

完全免费，MIT 开源协议，商用友好。cc4pm 本身不收取任何费用，你需要的只是一个 Claude Code 账号（或其他支持的 AI IDE 订阅）。所有 Skill、Agent 和工作流都可以自由修改和定制。欢迎在此基础上构建你自己的产品工作流。

不需要全部学。Skill 是 cc4pm 执行各类任务的最小单元，大多数时候你只需要记住几个高频命令就够了，比如 /bmad-create-prd、/bmad-brainstorming、/tdd、/code-review。

输入 /cc4pm-guide 后，AI 私教会根据你当前的任务主动推荐合适的 Skill，不需要你一一记忆。
