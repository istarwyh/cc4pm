FAQ

常见问题

用之前你可能想知道的事

课程完全在 Claude Code 终端里运行——没有视频，没有外部平台。克隆仓库，打开 Claude Code，输入 /cc4pm-guide，AI 会作为你的课程教练，带你逐节学习，边教边让你动手练习。

每节课都有具体的任务和命令要执行，学完一节就能立刻用在你自己的项目上。这是"边学边做"，不是"看完再做"。

共 26 节正式课 + 19 节补充课，分 5 个阶段，总学习时长约 2 小时。

你不需要一次学完——课程支持随时保存进度，下次继续。大多数人的节奏是：花 30 分钟完成阶段 1（基础入门），然后直接跳到和自己角色最相关的模块开始实战。

不需要。阶段 1-3（Claude Code 基础 + BMM + CIS + WDS）完全不需要编程基础，产品经理和创始人可以直接上手。阶段 4-5（工程工具链 + 高级编排）对开发者更友好，但非开发者也可以了解大概，知道"AI 能帮你做什么"。

需要的只是一个 Claude Code 账号，以及愿意改变工作方式的意愿。

"会用 AI"通常意味着知道怎么写 Prompt 得到一个好答案。cc4pm 教的是另一个层次：如何让 AI 代理自主完成一系列任务，并把结果衔接成产品交付物。

核心差别在于：Claude Code 的代理模式让 AI 可以自己探索代码库、自己运行测试、自己修复问题——你不是在聊天，而是在指挥一个能自主行动的团队。这套工作方式需要专门学习，cc4pm 的课程就是为此设计的。

课程完全免费，MIT 开源。你只需要：一个 Claude Code 账号（或 Cursor / Codex / OpenCode）和 Git。cc4pm 本身不收费，没有会员、没有付费内容，所有 26 节课和 200+ Skills 全部开放。

课程第 25 节是一个完整实战案例：从一个产品想法出发，经过 CIS 头脑风暴 → BMM 市场调研 → PRD 撰写 → WDS 设计 → 工程开发 → 上线交付，全程由 AI 代理执行，你来决策和审核。

学完之后，你应该能够：独立驱动这套流程处理你自己的项目，知道在每个阶段该调用哪个代理，以及如何在遇到问题时调整代理行为。一个人，做原来需要团队才能做的事。
