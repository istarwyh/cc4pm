**核心内容**

#### 什么是 cc4pm？

cc4pm（Claude Code Plugin Manager）是一个**生产就绪的 Claude Code 插件集合**。

它包含：
- **Agents**：专业化的子代理（planner、code-reviewer、tdd-guide 等）
- **Skills**：工作流定义和领域知识
- **Commands**：斜杠命令（/tdd、/plan、/e2e 等）
- **Hooks**：触发式自动化
- **Rules**：始终遵循的准则
- **MCP Configurations**：MCP 服务器配置

#### cc4pm 解决什么问题？

1. **效率问题**：无需从零开始，AI 已经掌握最佳实践
2. **一致性问题**：统一的工程标准和代码规范
3. **学习曲线问题**：交互式教学让你快速上手

**常见问题**

**Q: cc4pm 和普通的 Claude Code 有什么区别？**

A: 普通 Claude Code 需要你从零开始描述需求；cc4pm 提供了预定义的专业化工作流，让 AI 直接使用最佳实践。

**Q: 我需要付费吗？**

A: cc4pm 是开源免费的。
